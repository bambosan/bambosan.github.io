Mirai

About

A vibrant visual shader for MCBE that aims for the visuals to look realistic, that's it.

Technically, Mirai is a heavily modified vibrant visual pipeline. A total of 52 materials have been rebuilt and adjusted, where the material itself is a file used by the Render Dragon engine, containing shaders and configurations for the rendering pipeline.

Mirai is also a reboot of my old shader project that once shared the same name. The previous one focused on a different visual direction; this version is a complete reimagination. Instead of continuing the old style, now it's built around a realistic look and atmospheric feels.

Main Features

Volumetric Clouds

Real-time permeable volumetric clouds

Better Reflections

Cleaner and has more vertical tracing screen-space reflections

Atmospheric Scattering

Semi-physical atmosphere with aerial perspective for realistic fog and natural sky

Projected Water Caustics

Dynamic light patterns projected onto the seabed

AgX Tonemapping

Handles bright lights and colors more smoothly

Cirrus Clouds

High altitude planar cloud layers

Better Snell's Window

Now we can clearly see everything that is outside the water

Open Source

Mirai is built out of passion and shared freely with the community under the MIT License. Whether you want to see how the shader works under the hood or contribute to making it even better, the source code is free to explore.

Installation Guide

There are three ways to install it and no recommendation on which is the best; choose the one you like.

Using MB Loader app

  1. Download the app here
  2. Open the app, go to Settings
  3. Click Import addon, select the downloaded .mcpack file
  4. Back to Home and click Load with Draco
  5. Activate the resource pack
  6. Restart MB Loader

Using Levi Launcher app

  1. Download the app here
  2. Download the mod here (.so file)
  3. Open Levi Launcher, go to Manage Mods, click Add Mod and select the mod (.so file) from step 2
  4. Go to Content Management, click Resource Packs, click Import Resource Pack select the downloaded .mcpack file
  5. Click Launch
  6. Activate the resource pack
  7. Restart Levi Launcher

Using Ambient app

  1. Download the app here
  2. Open file manager, click the downloaded .mcpack file, open with Ambient
  3. Open the app, click Launch Game
  4. Activate the resource pack
  5. Restart Ambient
IMPORTANT: If combining Mirai with other packs, place it on top of them!
IMPORTANT: Volumetric clouds are NOT active by default. Check the pack's settings.
IMPORTANT: Volumetric clouds are heavy stuff. Set Vibrant Visual resolution to 50% or below before activating!

Installation Guide

There are several ways to install it. Once you have the .mcpack file, you can extract it and do whatever you want to install it, as long as it works and you know what you're doing. We'll show you the method using BRD (BetterRenderDragon) because it's considered the easiest.

  1. Install BRD using this script
  2. Open File Explorer and double click downloaded .mcpack file (this will open Minecraft)
  3. When Minecraft is open, go to video settings
  4. Enable "In-game graphics mode switching"
  5. Set graphics mode to "Fancy" (if not already)
  6. Activate the resource pack
  7. Join the world and change the graphics mode to vibrant visual
IMPORTANT: If combining Mirai with other packs, place it on top of them!
IMPORTANT: Volumetric clouds are NOT active by default. Check the pack's settings.
IMPORTANT: Volumetric clouds are heavy stuff. Set Vibrant Visual resolution to 50% or below before activating!

FAQ

Rain doesn't affect the visuals?

Rain contribution to lighting and atmosphere is not yet implemented. This feature is planned for future updates

Is Mirai performance friendly?

Without volumetric clouds the answer is yes, if volumetric cloud is enabled then the answer is no. It targets mid to high-end devices

Is this compatible with other resource packs (especially VV packs)?

Yes, you just have to put Mirai above all of them

Can I use this on servers?

Yes
