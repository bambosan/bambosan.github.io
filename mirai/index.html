About Mirai

Technically, Mirai is a heavily modified vibrant visual pipeline for Minecraft: Bedrock Edition. A total of 39 materials have been rebuilt and adjusted, where material itself is a file used by the RenderDragon engine, containing configurations for the rendering pipeline and shaders.

Mirai is a reboot of my old shader project that once shared the same name. The previous Mirai focused on a different visual direction, this version is a complete reimagination. Instead of continuing the old style, Mirai is now built around a realistic–vibrant look, aiming to keep Minecraft colorful while adding atmospheric feels, and cinematic lighting.

Compared to the default vibrant visuals, Mirai introduces features such as projected caustics, a semi-physically based atmosphere model, atmosphere based volumetric fog, AgX tonemapping, and lightweight 2D wisp clouds.

How to Install

Using MB Loader app:

  1. Download MB Loader from Play Store: here
  2. Open the app → go to Settings → Import addon → select the downloaded mirai.mcpack file → back to Home → Load with MBL2
  3. Activate the resource pack

Using Levi Launcher app:

  1. Download the app here: Levi Launcher
  2. Download the mod here: mtbinloader2
  3. Open the app → Manage Mods → Add Mod → select the mod file (.so) downloaded in step 2
  4. Go to Content Management → Resource Packs → Import Resource Pack → select the downloaded mirai.mcpack file
  5. Launch → activate the resource pack
⚠️   If the pack is already activated but nothing affected, please restart Minecraft.
